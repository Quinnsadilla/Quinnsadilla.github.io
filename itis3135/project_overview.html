Project Overview

Project Overview

  • A site for the Girls Who Code club at UNCC.
  • Intended users will be students who are in the club or would like to learn more about it.
  • The site will have information about the club, members, upcoming events, and how to join.

Page Design

  • Home
    • Purpose: The main landing page that welcomes new users and explains what the club is.
    • Audience: Primarily students who are club members or potential new members.
    • Content: Introduction to the club, welcome message, club's mission, a brief description of other pages, images of past events, and a join button that links to the join page.
    • Is user input required: No
    • Does it require data validation? N/A
    • Contains buttons, hyperlinks, or dropdowns? Button with a hyperlink to the join page.
    • Actions: Clicking the join button will navigate to the Join page.
    • Special notes: Includes images of past events.
  • Join
    • Purpose: Provide information on how to join the club and share club's social media links.
    • Audience: Potential new members interested in joining the club.
    • Content: Instructions for joining and a form for new members to fill out their name and interests.
    • Is user input required: Yes, through a form for new member interest.
    • Does it require data validation? Yes, fields like name, email, and reason for joining should be required.
    • Contains buttons, hyperlinks, or dropdowns? A "Submit" button for the form and hyperlinks to social media.
    • Actions: When the form is submitted, data is validated, and either a success message is displayed or users are prompted to correct errors.
    • Special notes: None
  • Members
    • Purpose: Showcase the council members and provide details about club leadership.
    • Audience: Current members, prospective members, and those interested in learning more about club leadership.
    • Content: List of council members with their roles, photos, and short bios. Option to contact council members directly through email links.
    • Is user input required: No
    • Does it require data validation? N/A
    • Contains buttons, hyperlinks, or dropdowns? Email links for contacting council members.
    • Actions: Clicking an email link opens the user’s default email client.
    • Special notes: None
  • Events
    • Purpose: Display upcoming and past events for the club.
    • Audience: Current members, prospective members, and the general public.
    • Content: List of upcoming events with descriptions, dates, times, and locations. Photos from past events and a link to RSVP for future events.
    • Is user input required: No
    • Does it require data validation? N/A
    • Contains buttons, hyperlinks, or dropdowns? RSVP button links to Niner Engage to RSVP.
    • Actions: Users can click an event to view more details or use the RSVP button to go to Niner Engage.
    • Special notes: None
  • Girls Who Code HQ
    • Purpose: Provide background information about the Girls Who Code organization and its mission, including resources and links to the national HQ website.
    • Audience: Anyone interested in learning more about the broader Girls Who Code movement.
    • Content: Information about the national Girls Who Code organization, its mission, and goals. Links to the official Girls Who Code website and related resources.
    • Is user input required: No
    • Does it require data validation? N/A
    • Contains buttons, hyperlinks, or dropdowns? Hyperlinks to the official Girls Who Code website.
    • Actions: Clicking on links will navigate users to the official Girls Who Code website.
    • Special notes: May include a FAQ or fun fact section.

Dynamic Functionality on the Website

  • Home
    • Image carousel: Showcases images of past club events, rotating automatically or allowing users to click through.
  • Join
    • Form Validation: Ensures that users fill out required fields in the form before submitting.
    • Form Submission Feedback: Shows a success message or alert when the form is submitted.
  • Members
    • Toggles: Clickable sections or toggles that expand and collapse to show more details about each member’s bio.
  • Events
    • Toggles: Clickable sections or toggles that expand and collapse to show more details about each event.
    • Alert: When navigating from the webpage to an external site, e.g., Canvas.
  • Girls Who Code HQ
    • Alert: When navigating from the webpage to an external site, e.g., Canvas.
